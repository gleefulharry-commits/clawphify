# Changelog

All notable changes to clawphify will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-04-13

### Added
- Initial release forked from graphify
- Pre-built OpenClaw v2026.3.22 framework knowledge graph
- Multi-agent configuration support:
  - `sessions_spawn` tool documentation
  - Agent-to-agent communication setup
  - Sub-agent limits configuration
  - Orchestrator pattern guidance
  - Cost optimization strategies
  - Nested spawning support
- CLI commands: `info`, `init`, `query`, `install`
- Configuration examples for common multi-agent setups
- Support for Claude Code, OpenClaw, Codex, OpenCode platforms
